# 📈 Interactive Sales Performance Dashboard (Tableau)

## 🎯 Project Objective
This project focuses on analyzing sales data to identify key growth drivers and business challenges. By utilizing Tableau's powerful visualization capabilities, I transformed raw transactional data into a strategic tool for business decision-making.

## 🚀 Key Insights
* **Regional Dynamics:** Identified the top-performing regions and those requiring strategic marketing intervention.
* **Product Profitability:** Analyzed which product categories contribute most to the bottom line.
* **Customer Trends:** Visualized sales cycles to understand peak purchasing periods.

## 🛠️ Technical Stack
* **Visualization:** Tableau Desktop
* **Data Processing:** Microsoft Excel (Data cleaning and pivot tables)
* **Analysis:** Trend analysis and year-over-year growth metrics.
